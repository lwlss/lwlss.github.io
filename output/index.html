    Testing Markdown

    One of the nice things about Nikola is that you have several different markup options for writing your articles. This article is written using Markdown.

    It's possible to write code:

    # Logistic growth model #
logist<-function(K,r,g,t) (K*g*exp(r*t))/(K+g*(exp(r*t)-1))

# Generalised logistic growth model #
# Solution of dx/dt = r*x*(1-(x/K)^v)
Glogist<-function(K,r,g,v,t) K/(1+(-1+(K/g)^v)*exp(-r*v*t))^(1/v)

    I'm not sure how to write math with MathJax.

    Inline: \( \frac{dx}{dt} = rx (1 - \frac{x}{K}) \). Display: $$ \frac{dx}{dt} = rx (1 - \frac{x}{K}) $$
